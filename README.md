# Visualino-binaries
<p>In this section you can download a compiled and ready to use version of Visualino. There are currently two major versions of Visualino: the official one based on the QtWebKit library (0.7.3) and the more modern one based on the QtWebEngine library (0.7.4). There are no differences between the two versions regarding the use of the program, since both include the same type of blocks. The QtWebKit library has been deprecated on Windows, so it is assumed that in the future it will disappear completely in favor of QtWebEngine. This is why there is no 64-bit version of Visualino for Windows. However, QtWebKit is still used on GNU/Linux systems, together with QtWebEngine.</p>
